# Serviço de Pagamento

Projeto prático em JavaScript para criar uma classe de serviço de pagamento.

## Funcionalidades

- Realizar pagamento
- Armazenar pagamentos em uma lista
- Classificar pagamentos acima de R$100,00 como categoria "cara"
- Classificar pagamentos de até R$100,00 como categoria "padrão"
- Consultar o último pagamento realizado

## Tecnologias utilizadas

- JavaScript
- Node.js
- Mocha
- Node Assert
